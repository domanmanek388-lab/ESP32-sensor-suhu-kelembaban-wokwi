# ESP32-sensor-suhu-kelembaban-wokwi
Project esp32 untuk mengukur dan menampilkan suhu dan kelembaban di wokwi.
Mengunakan dht22 sebagai sensor suhu dan kelembaban dan lcd 16x2 (12c) untuk menampilkan hasil yang didapat.

## Teknologi
- ESP32
- Arduino IDE
- Wokwi Simulator
